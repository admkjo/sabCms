       Settings

      1. This page makes it possible to edit the website name, site initials, site Title, Motto and Contact.

           

             2. An administrator can only edit and existing setting but can not create a new setting.


           

       

            - NOTE: A settings should only be edited only when the website name or the motto or the contact

                       or the initials or the title has been changed, else it should be left as it is
