             Adding Sub Pages


     1. Sub Pages are Pages that are under another page.




           2.  To add a Top Link,  click on the "Content" link.

               

             

             3. Search for The Page you want to create a sub page for in the table below                    

               and click on its corresponding "Add sub Page" Button.

                 


            4. Fill the forms given you.


                 -Title = the title of the  Sub page you are creating.


                 -Source = Is the source of Information you are adding to the Sub  Page.


                 -Content = the writings that will be seen on the Sub page.


                 -Show Page = if yes, means the Sub page you are creating will be shown on the website after saving


                                       else the Sub page will be created after saving but will not show on the website.


                 -Parent Category = tells you the page under which you are creating the new page.


                 -Resource = Used to add image to the Sub page.  An image of 500pixels X 350 pixels gives the best result  .

               


            3. Click on the "save new content" button  below, when done.


           4.The 'rest form' button when clicked will clear all details populated in the form.
